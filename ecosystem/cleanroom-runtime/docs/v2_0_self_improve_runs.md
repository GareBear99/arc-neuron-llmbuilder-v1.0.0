# v2.0 self-improvement runs

Adds concrete repo-side self-improvement planning and scaffolded sandbox runs.

Commands:
- `lucifer self-improve analyze`
- `lucifer self-improve plan [--target-key key]`
- `lucifer self-improve scaffold [--target-key key]`

The scaffold command creates `.arc_lucifer/self_improve_runs/<run_id>/` with a copied `worktree/` and a `manifest.json` that records tasks and validation steps.
