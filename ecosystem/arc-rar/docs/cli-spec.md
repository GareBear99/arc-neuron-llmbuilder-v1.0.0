# CLI spec

## Implemented now

- `arc-rar config print`
- `arc-rar config init`
- `arc-rar list <archive>`
- `arc-rar info <archive>`
- `arc-rar test <archive>`
- `arc-rar extract <archive> --out <dir>`
- `arc-rar create <output> <inputs...> --format zip|tar|tar-gz|seven-z`
- `arc-rar backend doctor`
- `arc-rar gui ...` file-IPC submission commands

## Partial / planned

- installer-driven `assoc install/remove`
- socket/named-pipe transport
- native GUI apps for macOS / Windows / Linux
