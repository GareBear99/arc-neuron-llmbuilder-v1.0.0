# Open Review Corridor and 3.0 License Direction

ARC-Neuron LLMBuilder is being developed with a public evidence trail from 1.0+ through the 2.x development corridor, while reserving the right to protect later 3.0 commercial artifacts under a more suitable license.

## Version policy

| Range | Meaning | Release status |
|---|---|---|
| 1.0+ | Public foundation and self-coded baseline | released / reviewable |
| 2.0 | Development bridge | open review path |
| 2.1-2.9 | Development corridor | no formal 2.9 release planned |
| 3.0 | Commercial base-model milestone | planned protected commercial direction |

## Why the path stays visible

The public route to 3.0 is intentionally preserved so developers, researchers, and reviewers can inspect how the system evolved: validators, benchmark logic, language configuration, training scaffolding, governance receipts, and promotion evidence.

## What changes at 3.0

The 3.0-era commercial outputs may be distributed under a separate protective license covering production weights, packaged systems, business-facing releases, and downstream commercial artifacts. This does not retroactively remove the license rights attached to earlier public releases.

## Language intelligence direction

ARC-Neuron is not designed to benchmark itself by parameter count alone. The project is pursuing higher utility per parameter through structured language configuration: 35-language lexical coverage, lineage, symbols, mathematics, transliteration, and root-meaning relationships.

The intended benchmark is intelligence density: useful output per parameter, backed by benchmark pass records, reproducible receipts, and low-weight local deployment behavior.
