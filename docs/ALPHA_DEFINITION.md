# Alpha Definition

Alpha is considered materially complete for this starter package when:
- the repo contains a doctrine, contract, promotion gates, and benchmark schema
- dataset families exist with non-trivial starter corpora
- benchmark families exist with non-trivial starter task suites
- repo validation runs cleanly
- repo capsule generation works against a real local folder
- baseline reporting scripts run cleanly
- export matrix and model ladder docs are present

This does **not** mean a frontier model has already been trained.
It means the package is ready for real data expansion, base-model selection, training, evaluation, and GGUF export work.
